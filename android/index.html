Botanic Gardens

This is a companion app for the Dunedin Botanic Gardens. The project is being developed by 5 Otago Polytechnic students, 2 to develop a web API, and 3 including myself to develop the android application.

  1. The app is built with google maps so the user is able to use all the standard google maps camera controls. Although we have bounded the camera to this area.
  2. Points of interest are display on the map.
  3. To help with screen clutter, the labels only expand when the user has zoomed in to a sufficient level.
  1. The user can select a collection marker to launch the collection details view.
  2. The collection details view displays the items in the collection, as well as a brief description of the collection.
  3. The item in the collection are also listed in an expandable list view.

Nearby Sites

This is the application I made for an individual assignment. The application uses the Google Places API to find businesses and tourist attractions around you. apk

  1. Use the slider to adjust the search radius.
  2. Nearby interest sites are listed.
  3. Select an item to view additional details.
